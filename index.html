HTML

  • HTML controls the structure of the web page
  • The head element contains information about the webpage not visible to the user
  • The body element represents the visible content shown to the user
  • Essential tags: !DOCTYPE html, head, title, body, header, footer
  • Text formatting tags: h1 to h6, p
  • Images tags: img src="image.jpg" alt="description"
  • List tags: ul, li, ol

CSS

  • CSS stands for Cascading Style Sheets and controls the styling of the web page
  • It controls the layout, colors, fonts, and overal look of web pages
  • It defines how elements like headers, paragraphs, buttons and other components should be displayed
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • Box-shadow adds shadow effect around elements
  • Margin and padding broders can be set by (margin: 10px;) or individually (margin-top, margin-right, margin-bottom, margin-left)
  • CSS tags: margin, padding, text-align: center, box-shadow

Git

  • GIT is a type of version control system (VCS) used to track changes in code during code development
  • GIT helps track, modyfy, and delete code during the entire application development
  • Git tags: Git Status, git checkout, git pull origin, git checkout -b (creates a branch), git add -A, git commit -m, git pull/push origin

JavaScript

  • JS is used to create dynamic and interactive web pages
  • JS works alongside HTML and CSS to create fully functional web pages
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • JS tags: if, for, const, let, var, console.log
